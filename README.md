🕒 Aplicativo PONTO
Sistema simples de controle de ponto eletrônico, desenvolvido para registrar horários de entrada e saída de colaboradores.

⚙️ Tecnologias
React Native – Interface do app

Node.js + Express – Backend da aplicação

MongoDB – Armazenamento de dados

🚀 Funcionalidades
Registro de entrada e saída

Histórico de batidas de ponto

Login de usuários

Interface simples e responsiva

▶️ Como usar
bash
Copiar
Editar
# Clone o repositório
git clone https://github.com/BMZadvogadosassociados/Aplicativo-PONTO

# Acesse a pasta do projeto
cd Aplicativo-PONTO

# Instale as dependências
npm install

# Inicie o app
npm start
